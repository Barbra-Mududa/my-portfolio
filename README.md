### Setup and installation

#### Install NodeJS
Verify you are running a latest node version with:
```
node -v
```

If node version 16, set it as default using:
```
nvm imtall 16
nvm use 16
nvm alias default 16
```

To view project on yur browser, run:
```
npm i / npm install
npm start
```
